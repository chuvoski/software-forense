#----------Software Forense----------#

Herramienta en desarrollo orientada al análisis y gestión de evidencia digital en dispositivos móviles.

-----------------------------------------------------------------------

#----------Descripción----------#

Software Forense es una aplicación diseñada para apoyar procesos de análisis técnico en entornos de investigación digital, enfocándose en:

- Gestión de evidencia multimedia
- Procesamiento y conversión de archivos
- Organización estructurada de datos extraídos
- Desarrollo modular para futuras funciones forenses

El proyecto se encuentra en fase de desarrollo activo.

-----------------------------------------------------------------------

#----------Características actuales----------#

° Interfaz gráfica desarrollada en Python  
° Estructura modular  
° Base para implementación de lógica forense  
° Preparado para integración con herramientas externas  

-----------------------------------------------------------------------

#----------Tecnologías utilizadas----------#

- Python 3.x
- CustomTkinter
- Git
- Entorno Linux (Pop!_OS)
